# Local development
Built with Visual Studio 2019 Community. Copy the `.env.sample` file to `.env` and change the variables:

* `EMAIL_PASSWORD`: Charie email password
* `DB_CONNECTION_STRING`: SQL Server (Azure SQL) connection string
